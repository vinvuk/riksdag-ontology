Riksdag Ontology

A formal OWL 2 DL ontology for the Swedish Riksdag, modeling members, documents, legislative processes, votes, and organizational structure. Aligned with ELI and W3C standards.

Ontology Modules

The ontology is organized into five modular OWL files, following the UK Parliament pattern of separating concerns into independent, composable modules.

Agency (Aktörer)

Organizations, people, roles, and time-bounded memberships in the Riksdag.

21 classes · 10 object properties · 10 datatype properties

Documents (Dokument)

Parliamentary and legislative document types with ELI FRBR layering.

49 classes · 33 object properties · 13 datatype properties

Procedure (Process)

The legislative process modeled as a directed graph of steps and routes.

28 classes · 11 object properties · 3 datatype properties

Voting (Votering)

Vote events, individual ballots, outcomes, and voting methods.

4 classes · 8 object properties · 8 datatype properties

Interface (Koppling)

Alignment axioms mapping to ELI, rpubl, Schema.org, W3C ORG, and Dublin Core.

0 classes · 0 object properties · 0 datatype properties
